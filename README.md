# vue-sample-app
  
ホスティングサービスなどで確認用に使うVue.jsプロジェクトです。  

## version
```
@vue/cli@4.2.2
```
  
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
